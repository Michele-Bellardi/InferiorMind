# Inferior Mind
## Descrizione del Progetto
Una versione semplificata del classico gioco "Master Mind" sviluppata in Flutter. Il gioco genera casualmente una sequenza segreta di 4 colori che il giocatore deve indovinare attraverso tentativi successivi.

## Logica di gioco e struttura dell'applicazione
In questa versione semplificata di Master Mind l'applicazione genera un codice casuale di colori e il giocatore ha a disposizione 4 bottoni, che premuti cambieranno il loro colore proponendo di volta in volta un diverso colore tra il set di quelli disponibili in modo da mostrare sul display la sequenza di colori che si vuole indovinare. 
Sotto ai bottoni è presente un bottone che premuto avvierà il processo di verifica del codice inserito dal giocatore.
Al disotto di quest'ultimo si trova il pulsante "reset gioco" utile per ripristinare il gioco e generare un nuovo codice segreto. 
L'ultimo elemento più in basso nella pagina è una casella di testo che mostra i seguenti messaggi in base alla situazione in cui il giocatore si trova.

## Per installare l'applicazione su Windows:
### Clonare la repository remota all'interno del computer:

git clone https://github.com/Michele-Bellardi/InferiorMind.git?authuser=0

### Avviare il progetto su Windows:
1. aprire il progetto con Android Studio o Visual Studio Code;
2. avviare un emulatore Android;
4. avviare il progetto.

## Per instalare l'applicazione su android:
Dopo aver aperto il progetto su Windows:
1. generare il file .apk da Android Studio;
2. trasferire il file generato all'interno della meoria del cellulare Android;
3. aprire il file .apk dal cellulare e seguire la procedura di installazione indicata;
4. aprire l'applicazione appena installata.
